
A basic 3D scene with movement options and selecting different objects. Possible to turn on wire-frame, triangle rasterization and backface culling on every object, has to be turned on programmatically though. For information about the object, use tab button to switch between all the different objects, which are - cube, sphere, tube (cylinder) and a plane for better orientation. 


W, S, A, D = movement

Tab = switching between objects

Mouse + right click = change camera view
